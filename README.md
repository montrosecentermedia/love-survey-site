# Houston Area L.O.V.E. Survey — Landing Page

Static HTML landing page for the Houston Area L.O.V.E. Survey (LGBTQ+ Opinions, Voices & Experiences).

## Setup

### 1. Add Images

Create an `images/` folder in the root directory and add the following files:

| Filename | Description | Source |
|---|---|---|
| `hero-desktop.png` | Wide banner (the magenta image with people + rainbow heart, ~1200x500) | Provided by marketing |
| `hero-mobile.png` | Square/tall banner for mobile (same design, ~600x600 or similar) | Provided by marketing |
| `logo-kinder.png` | Kinder Institute for Urban Research at Rice University logo | Provided by marketing |
| `logo-montrose.png` | The Montrose Center logo (color version on white/transparent bg) | Provided by marketing |
| `logo-chamber.png` | Greater Houston LGBTQ+ Chamber of Commerce logo | Provided by marketing |

### 2. Deploy to GitHub Pages

1. Open **GitHub Desktop**
2. Create a new repository (e.g., `love-survey`)
3. Copy this entire folder into the repo
4. Commit and push
5. Go to the repo on GitHub → **Settings** → **Pages**
6. Set source to `main` branch, root folder
7. Your page will be live at `https://yourusername.github.io/love-survey/`

## Features

- English / Español tab toggle (no page reload)
- Mobile-responsive design
- Accordion FAQ sections

- SEO meta tags and Open Graph tags
